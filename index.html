Installation guide

Follow these steps in order

1

Install Python 3.14

Pioneer requires Python 3.14 or newer. If you already have it, skip to step 2.

1. Go to python.org/downloads and download the latest 3.14 installer.
2. Run the installer.

⚠

Check "Add python.exe to PATH" at the bottom of the installer screen before clicking Install Now. The box is unchecked by default — if you miss it, Python won't be found.

3. After installation finishes, restart your computer.

4. Verify it worked — open Command Prompt and run:

python --version

You should see Python 3.14.x. If you get "not recognized", the PATH box was missed — uninstall and reinstall with it checked.

2

Download & extract the installer

Click the button at the top of this page to download installer.zip, or use the direct link:

installer.zip

Right-click the downloaded file → Extract All → choose a permanent folder, e.g. Documents\Pioneer-Installer. Don't delete this folder after install — you'll need it for updates.

3

Run the installer

Open the extracted folder. Hold Shift and right-click on empty space, then click Open PowerShell window here.

Type the following and press Enter:

python install.py
ℹ

The installer downloads ~3 GB total (Ollama + AI model). Leave the window open. If a download is blocked by your network, the installer will show you a direct link and wait for you to download it manually before continuing.

When you see INSTALLATION COMPLETE! you're done.

4

Launch Pioneer

Open the Pioneer folder that was created during install. Double-click start.bat.

Your browser will open automatically at http://localhost:8000

Your access code is in the Pioneer\.env file — open it with Notepad and look for ACCESS_CODE.

Keeping Pioneer up to date

Getting updates

Run the updater

When a new version is available, Pioneer will show an amber "update available" chip in the top-right of the interface. To install it:

Open PowerShell in your Pioneer-Installer folder and run:

python update.py

Type Y to confirm. Your data and settings are preserved automatically. The AI model and Ollama are never re-downloaded — only the app code updates.

Troubleshooting

Common issues

"python is not recognized as a command"
You missed the "Add python.exe to PATH" checkbox during Python installation. Uninstall Python from Control Panel, reinstall from python.org, and check that box before clicking Install Now. Restart your computer afterwards.
A download fails or gets blocked
The installer will print a direct GitHub link and pause. Open that link in your browser, download the file, move it into the installer folder, then press Enter in the installer window to continue. If GitHub itself is blocked, ask IT to whitelist github.com and objects.githubusercontent.com.
Installer is stuck or taking very long
The AI model is ~2 GB — on a slower connection this can take 20–30 minutes. Do not close the window. If it does fail, run python install.py again — it skips any steps that already completed.
Antivirus is blocking the installer
Add the Pioneer-Installer folder to your antivirus exclusion list, then re-run python install.py.
Pioneer opens but shows no answers (everything is "Needs SME Review")
The knowledge base is empty. Go to the Knowledge Base tab and upload the product datasheets (PDF or DOCX). Without datasheets loaded, Pioneer has nothing to answer from.
