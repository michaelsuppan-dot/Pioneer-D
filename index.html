Installation guide

Follow these steps in order

1

Install Python 3.12 or newer

Pioneer requires Python 3.12+. If you already have it, skip to step 2.

Go to python.org/downloads and download the latest installer, then run it.

⚠

Check "Add python.exe to PATH" at the bottom of the installer screen before clicking Install Now. The box is unchecked by default — if you miss it, Pioneer Setup won't find Python.

After installation finishes, restart your computer.

2

Download & run Pioneer Setup

Click the button at the top of this page to download Pioneer-Setup-1.2.2-NoBundle.exe, then double-click it to run.

ℹ

The setup wizard extracts the installer files and then automatically launches the Pioneer installer in a separate window. It downloads Ollama (~1.2 GB) and the AI model (~2 GB) from GitHub during installation.

⚠

Do NOT close the black installer window until it says INSTALLATION COMPLETE. Installation takes 15–30 minutes depending on your internet speed.

3

Wait for INSTALLATION COMPLETE

The black window will walk through 7 steps automatically:

1. Python version check
2. Download Pioneer app
3. Extract Pioneer app
4. Install Ollama
5. Download AI model (~2 GB — the slow step)
6. Set up Python environment
7. Set up Ollama model

When you see INSTALLATION COMPLETE! you're done.

ℹ

If a download is blocked by your network, the installer will show a direct link and wait. Download the file manually, move it into the installer folder shown on screen, then press Enter to continue.

4

Launch Pioneer

Open the Pioneer folder inside your Documents folder. Double-click start.bat.

Your browser will open automatically at http://localhost:8000

Your access code is in the Pioneer\.env file — open it with Notepad and look for ACCESS_CODE.

Keeping Pioneer up to date

Getting updates

Run the updater

When a new version is available, Pioneer will show an amber "update available" chip in the top-right of the interface. To install it:

Open PowerShell in your Pioneer-Installer folder and run:

python update.py

Type Y to confirm. Your data and settings are preserved automatically. The AI model and Ollama are never re-downloaded — only the app code updates.

Troubleshooting

Common issues

"python is not recognized as a command"
You missed the "Add python.exe to PATH" checkbox during Python installation. Uninstall Python from Control Panel, reinstall from python.org, and check that box before clicking Install Now. Restart your computer afterwards.
A download fails or gets blocked
The installer will print a direct GitHub link and pause. Open that link in your browser, download the file, move it into the installer folder shown on screen, then press Enter in the installer window to continue. If GitHub itself is blocked, ask IT to whitelist github.com and objects.githubusercontent.com.
Installer is stuck or taking very long
The AI model is ~2 GB — on a slower connection this can take 20–30 minutes. Step 7 (Ollama model setup) can also take a few minutes silently — this is normal. Do not close the window. If it does fail, re-run Pioneer-Setup-1.2.2-NoBundle.exe — completed steps are skipped automatically.
Cortex XDR or antivirus is blocking the installer
Pioneer Setup is an internal Motorola Solutions tool. If it is blocked, contact IT and ask them to add an exception for Pioneer-Setup-1.2.2-NoBundle.exe and the PioneerInstaller folder in your Documents. Pioneer Setup does not use cmd.exe — it launches Python directly, which should pass most behavioral threat policies.
Pioneer opens but shows no answers (everything is "Needs SME Review")
The knowledge base is empty. Go to the Knowledge Base tab and upload the product datasheets (PDF or DOCX). Without datasheets loaded, Pioneer has nothing to answer from.
