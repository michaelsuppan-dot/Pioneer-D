# Pioneer - RFP Response Engine

**Powered by Motorola Solutions / Avigilon / Pelco**

---

## Quick Start

1. **Double-click `start.bat`**
2. Browser opens automatically at http://localhost:8000
3. Enter access code (found in `.env` file)
4. Upload your RFP and process

---

## Configuration

Edit `.env` to change settings:
- `ACCESS_CODE` - Login password
- `VENDOR_NAME` - Your company name

---

## Folders

- **app/** - Application code
- **kb/** - Knowledge base (product datasheets)
- **data/** - Uploaded RFPs and generated outputs
- **models/** - AI model files
- **docs/** - Documentation and guides
- **internals/** - Configuration files and developer tools
- **distribution/** - Distribution system for sharing with team
