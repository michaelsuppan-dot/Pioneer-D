Capabilities

What Pioneer does

📂

Multi-format ingest

Drop in PDF, DOCX, or XLSX. Pioneer auto-detects structured Q&A workbooks and extracts requirements deterministically.

🧠

Local LLM (llama3.1:8b)

Runs on Ollama locally. Auto-tier launcher picks the best model for your machine. Nothing goes to the cloud.

📚

Product knowledge base

Pre-loaded with Avigilon datasheets — cameras, NVRs, appliances, access control. Drop in your own PDFs to extend it.

✅

Compliance tagging

Every response tagged Comply / Partial / Needs SME Review, with source citations back to the datasheet.

💾

Bundled installer

One 1.2 GB installer includes Ollama. SHA-256 verified. Handles SSL-inspection proxies and corporate firewalls.

🔁

In-place updates

Run python update.py when a new version drops. Data and settings preserved; the AI model is never re-downloaded.

Workflow

Three steps from RFP to draft response

01

Upload the RFP

Drop in the customer's document — PDF, Word, or Excel questionnaire.

02

Pioneer processes it

Requirements extracted, matched against the product KB, drafts written locally.

03

Review & export

SME reviews the flagged items, then exports the completed response.
